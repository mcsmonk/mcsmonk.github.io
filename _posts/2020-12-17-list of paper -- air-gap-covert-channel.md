# Air-gap covert channel

## list
This list is from Table 1 in \[[paper](https://arxiv.org/pdf/2012.06884.pdf)\]

- Electromagnetic
	- AirHopper: Bridging the Air-Gap between Isolated Networks and Mobile Phones using Radio Frequencies \[[paper](https://arxiv.org/pdf/1411.0237)\]
	- GSMem: Data Exfiltration from Air-Gapped Computers over GSM Frequencies \[[paper](https://www.usenix.org/system/files/sec15-paper-guri-update_v2.pdf)\]
	- USBee: Air-Gap Covert-Channel via Electromagnetic Emission from USB \[[paper](https://arxiv.org/pdf/1608.08397)\]
	- AIR-FI: Generating Covert Wi-Fi Signals from Air-Gapped Computers \[[paper](https://arxiv.org/pdf/2012.06884.pdf)\]
- Magnetic
	- MAGNETO: Covert Channel between Air-Gapped Systems and Nearby Smartphones via CPU-Generated Magnetic Fields \[[paper](https://arxiv.org/pdf/1802.02317)\]
	- ODINI : Escaping Sensitive Data from Faraday-Caged, Air-Gapped Computers via Magnetic Fields \[[paper](https://arxiv.org/pdf/1802.02700)\]
- Electric
	- PowerHammer: Exfiltrating Data from Air-Gapped Computers through Power Lines \[[paper](https://arxiv.org/pdf/1804.04014)\]
- Acoustic
	- Fansmitter: Acoustic Data Exfiltration from (Speakerless) Air-Gapped Computers \[[paper](https://arxiv.org/pdf/1606.05915)\]
	- DiskFiltration: Data Exfiltration from Speakerless Air-Gapped Computers via Covert Hard Drive Noise \[[paper](https://arxiv.org/pdf/1608.03431)\]
	- On Covert Acoustical Mesh Networks in Air \[[paper](https://arxiv.org/pdf/1406.1213)\]
	- MOSQUITO: Covert Ultrasonic Transmissions between Two Air-Gapped Computers using Speaker-to-Speaker Communication \[[paper](https://arxiv.org/pdf/1803.03422)\]
	- POWER-SUPPLaY: Leaking Data from Air-Gapped Systems by Turning the Power-Supplies Into Speakers \[[paper](https://arxiv.org/pdf/2005.00395)\]
	- CD-LEAK: Leaking Secrets from Audioless Air-Gapped Computers Using Covert Acoustic Signals from CD/DVD Drives \[[paper](https://ieeexplore.ieee.org/stamp/stamp.jsp?tp=&arnumber=9202547)\]
- Thermal
	- BitWhisper: Covert Signaling Channel between Air-Gapped Computers using Thermal Manipulations \[[paper](https://arxiv.org/pdf/1503.07919)\]
	- HOTSPOT: Crossing the Air-Gap Between Isolated PCs and Nearby Smartphones Using Temperature \[[paper](https://ieeexplore.ieee.org/stamp/stamp.jsp?tp=&arnumber=9108874)\]
- Optical
	- LED-it-GO: Leaking (a lot of) Data from Air-Gapped Computers via the (small) Hard Drive LED \[[paper](https://arxiv.org/pdf/1702.06715)\]
	- An optical covert-channel to leak data through an air-gap\[[paper](https://ieeexplore.ieee.org/stamp/stamp.jsp?tp=&arnumber=7906933)\]
	- CTRL-ALT-LED: Leaking Data from Air-Gapped Computers via Keyboard LEDs \[[paper](https://arxiv.org/pdf/1907.05851)\]
	- xLED: Covert Data Exfiltration from Air-Gapped Networks via Router LEDs \[[paper](https://arxiv.org/pdf/1706.01140)\]
	- aIR-Jumper: Covert Air-Gap Exfiltration/Infiltration via Security Cameras & Infrared (IR) \[[paper](https://arxiv.org/pdf/1709.05742)\]
- Vibrations
	- AiR-ViBeR: Exfiltrating Data from Air-Gapped Computers via Covert Surface ViBrAtIoNs \[[paper](https://arxiv.org/pdf/2004.06195)\]
